# Clsalarm - An alarm clock for classes (originally 华育铃声)

During the COVID-19 pandemic, many students have been studying online. This is
a simple app that reminds students of the classes and activities they need to
attend. This is originally designed for our school, so it is also known as "华育铃声".

## Installation

```bash
python3 -m pip install git+https://github.com/acciochris/clsalarm
```

## Usage

Prerequisites:
- Python 3.6 and above

```bash
python3 -m alarm -c <configuration file>
```
